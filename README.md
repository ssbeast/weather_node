# weather_node
Using Nodejs and Expressjs

Used WeatherStack and MapBox API in project!

To run the project:

1. Clone this repo using `git clone https://github.com/ssbeast/weather_node`
2. Run `npm install`
3. Run `npm start`

## Live demo at https://ssbeast-weather.herokuapp.com/
